– registerSubclass: required method

Register a class that conforms to AGObjectMappingProtocol as a valid mappable/unmapple class. Registering a class will make this class ready for mapping/unmapping operations. Under the hood, registering a class will try to instanciate an object of this class and call AGObjectMapping configuration methods.

- (BOOL)registerSubclass:(nonnull Class __unsafe_unretained)newClass

The class to register.

YES if a class has been registered. NO otherwise.
